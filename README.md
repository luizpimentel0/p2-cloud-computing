## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Create a .env file in root directory

```bash
DATABASE_URL="mysql://user:pass@serverurl:3306/table"
```
